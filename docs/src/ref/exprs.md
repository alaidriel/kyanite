# Expressions

### Literals

`"Hello, world!"`, `3`, `3.14`, `true`

### Identifiers

Identifiers are a sequence of one or more characters, where the first character may not be a digit `0-9`.

`foo`, `bar`, `baz`

### Calls

`foo()`, `foo(1, 2, 3)`

### Binary Operators

`+`, `-`, `*`, `/`, `==`, `!=`, `>`, `<`, `>=`, `<=`

Logical operators are not currently supported.

### Unary Operators

`-`, `!`

### Grouping

`(1 + 2) * 3`
